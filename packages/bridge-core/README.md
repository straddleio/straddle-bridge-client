# `@straddlecom/bridge-core`

## Overview

`@straddlecom/bridge-core` contains shared core utilities and types used by `@straddlecom/bridge-js` and `@straddlecom/bridge-react`. While it's a public package, it's primarily intended for internal use and not for direct consumption in applications.

## Installation

If you need to install this package directly (though it's generally not necessary), you can do so using npm:

```bash
npm install @straddlecom/bridge-core
```

Or using yarn:

```bash
yarn add @straddlecom/bridge-core
```

## Note

It's recommended to use either `@straddlecom/bridge-js` or `@straddlecom/bridge-react` for integrating Straddle's services into your applications.
